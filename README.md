<br>

# Installation

#### Install 
``` bash
# init server
npm init -y
npm install express cors body-parser
npm install --save-dev typescript @types/node @types/express ts-node nodemon
npx tsc --init

# init client
yarn create react-app client --template typescript
cd client
npm install axios
```
<br>

#### Start the server :
```bash
cd server
npm run dev
```
<br>

#### Start the client :
```bash
cd client
npm start
```
<br>

#### Open the project in your brower :
```bash
http://localhost:3000
```
